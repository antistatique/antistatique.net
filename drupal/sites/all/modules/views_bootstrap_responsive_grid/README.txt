INTRODUCTION
------------
Views Bootstrap Responsive Grid module enable you to take advantage of the
Bootstrap 3 Grid System columns reset
(http://getbootstrap.com/css/#grid-responsive-resets).
With this module you can define the number of items per row for each
breakpoints (http://getbootstrap.com/css/#grid-options). All of this within
the great configuration settings of the View module.

REQUIREMENTS
------------
 * Views module (http://drupal.org/project/views)

 * Your theme should include the Bootstrap3 CSS Grid System.
   This module don't provide any CSS and there is no hard dependency
   on a specific drupal theme.


RECOMMENDED MODULES
-------------------
 * Bootstrap theme (https://www.drupal.org/project/bootstrap)

INSTALLATION
------------
 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.
